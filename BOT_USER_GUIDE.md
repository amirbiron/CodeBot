## 🚀 התחלה מהירה

### 1. התחלת שיחה עם הבוט
פשוט שלח:
```
/start
```

הבוט יברך אותך ויציג **מקלדת קבועה** בתחתית המסך עם כל הכפתורים הראשיים.

### 2. הוספת קוד ראשון
1. לחץ על כפתור **➕ הוסף קוד חדש**
2. הבוט יבקש ממך לשלוח את הקוד
3. שלח את הקוד שלך
4. הבוט יזהה אוטומטית את שפת התכנות וישמור

**דוגמה:**
```python
def hello_world():
    print("Hello, World!")
    return "success"
```

### 3. קבלת עזרה
לחץ על כפתור **ℹ️ הסבר על הבוט** בתפריט התחתון, או שלח:
```
/help
```

## 🎯 התפריט הראשי

אחרי `/start`, יופיעו לך **כפתורי מקלדת קבועים** בתחתית המסך:

### שורה 1:
- **🗜️ יצירת ZIP** - יצירת ארכיון ZIP מהקבצים שלך
- **➕ הוסף קוד חדש** - הוספת קוד חדש למערכת

### שורה 2:
- **📚 הצג את כל הקבצים שלי** - ניהול וצפייה בקבצים
- **🔧 GitHub** - אינטגרציה מלאה עם GitHub

### שורה 3:
- **⚡ עיבוד Batch** - ניתוח מרובה של קבצים
- **📥 ייבוא ZIP מריפו** - ייבוא פרויקט שלם

### שורה 4:
- **☁️ Google Drive** - אינטגרציה עם Google Drive
- **ℹ️ הסבר על הבוט** - מדריך עזרה מפורט

**💡 טיפ:** הכפתורים האלה תמיד זמינים - אין צורך לזכור פקודות!

## ➕ הוספת קוד חדש

### איך להוסיף קוד?

1. **לחץ על כפתור ➕ הוסף קוד חדש** בתפריט הראשי
2. הבוט ישאל איך תרצה להוסיף:
   - **📝 שלח קוד ישירות** - פשוט שלח את הקוד
   - **📎 שלח קובץ** - העלה קובץ מהמחשב
   - **📋 העתק והדבק** - העתק קוד ארוך

3. **שלח את הקוד שלך**
4. הבוט יזהה אוטומטית את:
   - שפת התכנות
   - שם קובץ מתאים
   - סוג הקוד

5. **אישור ושמירה**
   - הבוט יציג תצוגה מקדימה
   - אפשרות להוסיף תגיות
   - אפשרות להוסיף תיאור

### תכונות:
- ✅ זיהוי שפת תכנות אוטומטי (20+ שפות)
- ✅ ניקוי קוד אוטומטי (הסרת BOM, NBSP, ZWSP)
- ✅ תמיכה בקבצים גדולים (עד 50MB)
- ✅ הדגשת תחביר צבעונית
- ✅ שמירת גרסאות אוטומטית

## 📚 ניהול קבצים

### תפריט הקבצים

**לחץ על כפתור 📚 הצג את כל הקבצים שלי**

תקבל תפריט עם אפשרויות:

#### 🔎 חפש קובץ
חיפוש מתקדם לפי:
- **שם קובץ** - `name:utils`
- **שפת תכנות** - `lang:python` או `lang:js`
- **תגית** - `tag:api` או `tag:repo:owner/name`
- **שילוב** - `lang:python name:util tag:api`

**דוגמאות:**
```
utils.py
lang:javascript
tag:repo:myproject/frontend
name:test lang:python
```

---

#### 🗂 לפי ריפו
רשימת כל הריפוזיטוריז שלך:
- כל ריפו מוצג עם מספר הקבצים
- לחיצה על ריפו מציגה את הקבצים שלו
- עימוד אוטומטי

---

#### 📂 קבצים גדולים
קבצים מעל 500 שורות:
- טעינה חכמה
- צפייה בחלקים
- הורדה מהירה

---

#### 📁 שאר הקבצים
כל הקבצים הרגילים:
- מיון לפי תאריך
- עימוד (10 קבצים בעמוד)
- כפתורי ניווט ⬅️ / ➡️

---

#### ⭐ מועדפים
קבצים שסימנת כמועדפים:
- גישה מהירה לקבצים חשובים
- סימון בכוכב ⭐

---

#### 🗑️ סל מיחזור
קבצים שמחקת:
- אפשרות שחזור
- מחיקה סופית
- ניקוי הסל

---

### תפריט קובץ בודד

כשלוחצים על קובץ, מקבלים תפריט עם כפתורים:

- **👁️ הצג** - תצוגת הקוד עם הדגשת תחביר
- **✏️ ערוך** - עריכת הקוד
- **📝 שנה שם** - שינוי שם הקובץ
- **📚 היסטוריה** - כל הגרסאות הקודמות
- **📥 הורד** - הורדה למחשב
- **🗑️ העבר לסל** - מחיקה (עם אפשרות שחזור)
- **⭐ הוסף למועדפים** / **💔 הסר ממועדפים**
- **🏷️ נהל תגיות** - הוספה/הסרה של תגיות
- **📊 נתח קובץ** - ניתוח מפורט
- **🌐 שתף** - שיתוף בפלטפורמות שונות

---

### עריכת קובץ

1. לחץ על **✏️ ערוך** בתפריט הקובץ
2. שלח את הקוד המעודכן
3. הבוט:
   - ✅ שומר גרסה קודמת אוטומטית
   - ✅ מציג diff בין הגרסאות
   - ✅ מאפשר ביטול שינויים

---

### מחיקת קובץ

1. לחץ על **🗑️ העבר לסל**
2. מסך אישור יופיע
3. הקובץ עובר לסל מיחזור
4. אפשר לשחזר תוך 30 יום

**💡 טיפ:** מחיקה היא לא סופית! תמיד אפשר לשחזר מהסל.

## 📚 ניהול גרסאות

### צפייה בהיסטוריה

1. בחר קובץ מהרשימה
2. לחץ על **📚 היסטוריה**
3. תקבל רשימת גרסאות:

**תצוגת גרסאות:**
```
📚 היסטוריית script.py:

v3 - 15/08/2025 14:30 (נוכחית)
✏️ הוספת טיפול בשגיאות
[👁️ הצג] [🔄 שחזר]

v2 - 14/08/2025 10:15
✏️ שיפור ביצועים
[👁️ הצג] [🔄 שחזר] [⚖️ השווה]

v1 - 13/08/2025 08:00
✏️ גרסה ראשונית
[👁️ הצג] [⚖️ השווה]
```

### כפתורים בהיסטוריה:
- **👁️ הצג** - צפייה בגרסה זו
- **🔄 שחזר** - שחזור לגרסה זו
- **⚖️ השווה** - השוואה עם גרסה אחרת

---

### שחזור גרסה קודמת

1. בחר גרסה מההיסטוריה
2. לחץ על **🔄 שחזר**
3. הבוט יציג:
   - ✅ תצוגה מקדימה של השחזור
   - ⚖️ diff בין הגרסאות
   - ⚠️ אזהרה שהגרסה הנוכחית תישמר
4. אשר שחזור

**תכונות:**
- ✅ הגרסה הנוכחית נשמרת אוטומטית
- ✅ אפשר לבטל את השחזור
- ✅ ניתן לשחזר כל גרסה בהיסטוריה

---

### השוואת גרסאות

1. בחר גרסה מההיסטוריה
2. לחץ על **⚖️ השווה**
3. בחר גרסה שנייה להשוואה
4. תקבל diff מפורט:
   - **➕ שורות שנוספו** (ירוק)
   - **➖ שורות שהוסרו** (אדום)
   - **🔄 שורות ששונו** (כתום)
   - **📊 סטטיסטיקה** (כמה שורות, אחוזים)

## 📊 ניתוח קוד

### ניתוח קובץ בודד

1. בחר קובץ מהרשימה
2. לחץ על **📊 נתח קובץ**
3. הבוט יריץ ניתוח מפורט

**דוח ניתוח כולל:**

#### 📏 מדדי גודל:
- סך הכל שורות
- שורות קוד (LOC)
- שורות הערות
- שורות ריקות
- גודל קובץ (KB/MB)

#### 🔧 מבנה קוד:
- מספר פונקציות
- מספר מחלקות
- עומק קינון מקסימלי
- מורכבות ציקלומטית
- ציון איכות (0-100)

#### 📝 פונקציות שנמצאו:
```
def process_data(input: str) -> dict
  📍 שורה 45
  📊 16 שורות
  ⚡ מורכבות: 3

class DataHandler:
  📍 שורה 78
  🔧 4 מתודות
```

#### 💡 הצעות לשיפור:
- ✅ הוסף הערות למתודות
- ⚠️ פונקציה ארוכה מדי (שורה 120)
- 🔴 קוד כפול זוהה (שורה 45, 89)
- 💚 איכות קוד טובה (ציון: 87)

**כפתורים נוספים:**
- **✅ בדוק תקינות** - בדיקת תחביר
- **📋 העתק דוח** - העתקת הדוח
- **📥 הורד דוח** - הורדת הדוח כ-PDF

---

### בדיקת תקינות

1. בחר קובץ
2. לחץ על **✅ בדוק תקינות**
3. הבוט יבדוק:
   - ✅ תחביר נכון
   - ✅ imports קיימים
   - ✅ משתנים מוגדרים
   - ✅ לוגיקה תקינה

```
🐙 תפריט GitHub

🔑 הגדר טוקן GitHub
📁 בחר ריפו

📤 העלה קובץ לריפו

🔄 Pull Requests (יצירה/מיזוג)
🔔 התראות חכמות
🏷️ נקודת שמירה (Git Checkpoint)
💾 גיבוי מלא מריפו
🔍 נתח ריפו
✅ בדוק תקינות ריפו
```

### הגדרה ראשונית

#### 1️⃣ הגדרת טוקן GitHub

1. לחץ על **🔑 הגדר טוקן GitHub**
2. הבוט יסביר איך ליצור טוקן
3. עבור ל-[GitHub Settings > Tokens](https://github.com/settings/tokens)
4. צור Personal Access Token חדש:
   - בחר **Classic Token**
   - הרשאות: `repo`, `gist`, `delete_repo` (אופציונלי)
   - תוקף: 90 יום (או יותר)
5. העתק את הטוקן
6. שלח לבוט

**🔒 אבטחה:**
- ✅ הטוקן מוצפן במסד הנתונים (Fernet)
- ✅ לא נשמר בלוגים
- ✅ אפשרות למחיקה בכל עת (כפתור 🗑️ מחק טוקן)
- ✅ לא נשלח לשירותים חיצוניים

---

#### 2️⃣ בחירת Repository

1. לחץ על **📁 בחר ריפו**
2. תקבל רשימת הריפוזיטוריז שלך:
   - עימוד (10 ריפוזים בעמוד)
   - כפתורי ⬅️ / ➡️ לניווט
   - חיפוש לפי שם
3. לחץ על הריפו שבו תרצה לעבוד
4. הריפו נשמר למשימות עתידיות

**💡 טיפ:** ניתן לשנות ריפו בכל עת מהתפריט!

---

### דפדפן קבצים ופעולות GitHub

#### 📥 הורדת קבצים

1. לחץ על **📥 הורד קובץ מריפו**
2. נווט בתיקיות:
   - 📂 כפתורי תיקיות
   - 📄 כפתורי קבצים
   - ⬆️ למעלה
   - 🏠 חזרה לשורש
3. לחץ על קובץ להורדה
4. הקובץ יישלח אליך

**תכונות:**
- ✅ Breadcrumbs לניווט מהיר
- ✅ עימוד (10 פריטים בעמוד)
- ✅ הורדת תיקייה כ-ZIP (כפתור 📦)

---

#### 📤 העלאת קבצים

1. לחץ על **📤 העלה קובץ לריפו**
2. בחר מקור:
   - **📝 מהקבצים השמורים** - קבצים שיש בבוט
   - **📎 שלח קובץ חדש** - העלה קובץ חדש
3. בחר יעד (נתיב בריפו)
4. אשר העלאה

---

#### 🗑️ מחיקת קבצים

1. לחץ על **🗑️ מחק קובץ מריפו**
2. נווט לקובץ
3. לחץ עליו
4. בחר מצב:
   - **מחיקה ישירה** - מחיקה מיידית
   - **🔒 מחיקה בטוחה** - יצירת PR במקום

**💡 מחיקה בטוחה מומלצת!** - יוצרת PR לסקירה לפני מחיקה סופית.

---

#### ✅ בחירה מרובה

1. לחץ על **✅ בחר מרובים**
2. סמן קבצים מהרשימה (✅ / ☐)
3. פעולות זמינות:
   - **📦 הורד נבחרים כ-ZIP**
   - **🗑️ מחק נבחרים**
   - **🔗 שתף קישורים**

---

### Pull Requests

#### 🆕 יצירת PR חדש

1. לחץ על **🔄 Pull Requests**
2. בחר **🆕 צור PR מסניף**
3. בחר branch מקור (head) - עימוד אוטומטי
4. אשר יצירה
5. PR נוצר ל-branch ברירת המחדל

---

#### 🔀 מיזוג PR פתוח

1. לחץ על **🔀 מזג PR פתוח**
2. רשימת PRים פתוחים (עם עימוד)
3. בחר PR
4. אשר מיזוג
5. Merge מתבצע אוטומטית

---

### התראות חכמות

#### הגדרת התראות

1. לחץ על **🔔 התראות חכמות**
2. **⚙️ הגדרות:**
   - 🔔 הפעל/כבה התראות
   - 📋 בחר סוגים: PRs / Issues / שניהם
   - ⏱️ תדירות: 30 שנ׳ / 2 דק׳ / 5 דק׳ / 15 דק׳
3. **🔔 בדוק עכשיו** - בדיקה מיידית

**מקבלים התראות על:**
- ✅ PRs חדשים
- ✅ Issues חדשים
- ✅ עדכונים ל-PRs/Issues קיימים
- ✅ קישורים ישירים לכל אירוע

---

### נקודת שמירה (Git Checkpoint)

1. לחץ על **🏷️ נקודת שמירה בגיט**
2. הבוט יוצר:
   - Tag: `checkpoint-YYYYMMDD-HHMMSS`
   - אם נכשל → Branch במקום

**שחזור מנקודת שמירה:**
```bash
# משוך tags
git fetch --tags

# מעבר ל-tag
git checkout tags/checkpoint-20250815-143000

# צור branch חדש
git checkout -b restore-from-checkpoint
```

```
   🔴 שגיאות תחביר
   🔴 מורכבות גבוהה מדי
```

#### 📈 גרפים וסטטיסטיקות:
- התפלגות ציונים
- בעיות נפוצות
- שפות בשימוש
- המלצות לשיפור
