## 📝 פוסט #1: Live Preview עם Split View

**מכירים את זה שאתם עורכים Markdown ועובדים עם Alt+Tab כל 10 שניות בין העורך לדפדפן רק כדי לראות איך זה נראה?**

נמאס לכם לרענן את הדפדפן כל פעם ולאבד את הפוקוס?

בדיוק בשביל זה הוספתי את **Live Preview עם Split View** ל-CodeBot.

זה לא עוד עורך Markdown רגיל. זה סביבת עבודה מלאה שחוסכת לכם שעות:

• **רינדור בזמן אמת** - כל שינוי שאתם עושים מתעדכן מיידית בתצוגה. אפס רענונים, אפס המתנה 💨

• **Split View חכם** - המסך מתחלק לשניים: משמאל העורך, מימין התוצאה. גוררים את המפריד ומתאימים לפי הצורך 🎨

• **Debounce של 500ms** - העדכונים מתוזמנים בצורה חכמה, לא מעמיסים על הדפדפן, פשוט עובד חלק

• **תמיכה בכל מה שצריך**:
  - דיאגרמות Mermaid (Flowcharts, Sequence, Gantt - הכל!)
  - נוסחאות מתמטיות עם KaTeX
  - Syntax Highlighting עם highlight.js
  - Task Lists עם תיבות סימון אינטראקטיביות
  - Admonitions (note, warning, tip, danger)
  - תוכן עניינים אוטומטי 📊

• **קיצורי מקלדת** - Shift+Ctrl+Enter (Win) או Shift+Cmd+Enter (Mac) להחלפת Preview

• **מצבים מרובים** - Desktop עם Split, Mobile עם טאבים. תמיד נוח

• **Client-Side Rendering** - רוב העבודה נעשית בדפדפן שלכם, מהיר כמו שיער 🚀

• **Server Fallback** - יש משהו מורכב? השרת קופץ לעזרה אוטומטית

הכי טוב? זה נשמר ב-localStorage - הגודל שבחרתם, המצב שהעדפתם. פעם הבאה שאתם נכנסים, הכל כבר מחכה.

✅ מפתחים שכותבים תיעוד
✅ Technical Writers
✅ כל מי שעובד עם Markdown ונמאס לו מהמעברים

## 📝 פוסט #2: Incident Storyboard - ניהול תקריות

**כמה פעמים קרה לכם ש-Production בוער, הכל נפתר אחרי 3 שעות של חפירות, ואז המנהל שואל "אז מה קרה?" ואתם מתחילים לגמגם?**

## 📝 פוסט #3: תזכורות לפתקים (Sticky Notes Reminders)

**מכירים את זה שאתם משאירים TODO ב-Markdown, סוגרים את הקובץ, ופשוט שוכחים?**

נמאס לכם לפספס משימות חשובות כי הן קבורות באיזה קובץ תיעוד מלפני שבוע?

בדיוק בשביל זה הוספתי **תזכורות לפתקים דביקים** ל-CodeBot.

לא מדובר בתזכורות רגילות. זה מערכת שמשתלבת ישירות עם קבצי Markdown שלכם:

• **Sticky Notes על Markdown** - יוצרים פתק צף ישירות על קובץ ה-Markdown. אפשר לשים אותו בדיוק במקום הרלוונטי בקוד 📌

• **Presets מוכנים** - צריכים תזכורת מהר?
  - `1h` - בעוד שעה
  - `3h` - בעוד 3 שעות
  - `1d` / `24h` - מחר באותה שעה
  - `1w` / `7d` - בעוד שבוע
  - `today-21` - היום בשעה 21:00
  - `tomorrow-09` - מחר ב-9 בבוקר ⏰

• **זמן מותאם אישית** - רוצים תזכורת ליום ספציפי? פורמט `YYYY-MM-DDTHH:MM` ותמיכה מלאה ב-datetime-local

• **תמיכה באזורי זמן** - עובדים עם צוות גלובלי? מעבירים `tz` parameter (למשלה `Asia/Jerusalem`)

• **דחיית תזכורת (Snooze)** - קיבלתם את התזכורת אבל עדיין לא מוכנים? Snooze ל-60 דקות בלחיצה 💤

• **Dashboard מרכזי** - רשימה של כל התזכורות הממתינות, הבאה בתור, וספירה כוללת. שליטה מלאה

• **התראות חכמות** - התזכורת מגיעה בדיוק בזמן, עם תצוגה מקדימה של הטקסט והקישור ישר לקובץ

• **מעקב סטטוס** - `pending`, `sent`, `snoozed` - תמיד יודעים מה הסטטוס של כל תזכורת

למי זה מתאים?
✅ מפתחים עם TODO ארוך
✅ Tech Leads שצריכים לזכור Code Reviews
✅ כל מי שמנהל משימות דרך Markdown
✅ אנשים ש"אני אחזור לזה אחר כך" הופך אצלם ל"שכחתי מזה לגמרי"

זה הפתרון שלא ידעתם שאתם צריכים. מערכת תזכורות שמשתלבת עם מה שכבר עושים, לא עוד אפליקציית TODO נפרדת.

## 📝 פוסט #4: Workspace Kanban - ניהול משימות

**מכירים את זה שיש לכם 20 קבצים פתוחים, אתם לא זוכרים מה בתהליך ומה חיכה, והכל פשוט... chaos?**

נמאס לכם לעבוד עם Jira/Trello רק כדי לעקוב אחרי הקבצים שלכם?

בדיוק בשביל זה הוספתי **Workspace Kanban** ל-CodeBot.

זה לא עוד לוח משימות. זה Kanban שמשתלב ישירות עם הקבצים שאתם עובדים איתם:

• **3 עמודות פשוטות**:
  - 📋 **לטיפול** (To Do) - קבצים שחיכו לכם
  - ⚙️ **בתהליך** (In Progress) - מה שאתם עובדים עליו עכשיו
  - ✅ **הושלם** (Done) - סיימתם? תזיזו לכאן

• **Drag & Drop חלק** - גוררים קבצים בין העמודות עם העכבר. פשוט כמו שצריך להיות 🖱️

• **קיצורי מקלדת**:
  - `Shift+1` - העבר ל"לטיפול"
  - `Shift+2` - העבר ל"בתהליך"
  - `Shift+3` - העבר ל"הושלם"
  - עובדים מהמקלדת? אין בעיה ⌨️

• **ספירה בזמן אמת** - בראש כל עמודה יש ספירה שמתעדכנת אוטומטית. תמיד יודעים כמה משימות יש בכל שלב

• **גרירה בין אוספים** - רוצים להעביר קובץ מאוסף לאוסף? פשוט גוררים. זה עובד גם בין Workspace לאוספים אחרים 🔄

• **תמיכה במובייל** - גרירה במגע, responsive לגמרי. עובד מושלם גם בטלפון

• **התראות נגישות** - `aria-live` announcements למי שמשתמש בקורא מסך. Accessibility First ♿

• **ספריית SortableJS** - מימוש גרירה מקצועי וחלק, לא איזה hack מהבית

• **עדכון אוטומטי** - משנים מצב? ה-API מתעדכן מיידית ב-background, אפס המתנה

למי זה מתאים?
✅ מפתחים שצריכים לעקוב אחרי קבצים מרובים
✅ Tech Writers עם הרבה מסמכים בתהליך
✅ כל מי שאוהב Kanban אבל שונא כלים כבדים
✅ אנשים שרוצים ארגון פשוט וויזואלי

זה הפתרון לכל מי ש-VS Code Tabs זה לא מספיק בשבילו. ניהול משימות ישירות על הקבצים, אפס friction.

## 📝 פוסט #15: Markdown Toolbar - עריכה מהירה

**מכירים את זה שאתם כותבים Markdown וצריכים לזכור את כל הסינטקס? `**bold**` או `__bold__`? `#` או `##`?**

נמאס לכם לחפש בגוגל "markdown syntax" כל פעם?

בדיוק בשביל זה הוספתי **Markdown Toolbar** ל-CodeBot.

זה לא סרגל כלים רגיל. זה עוזר אישי לעריכת Markdown:

• **עיצוב טקסט בקליק**:
  - **Bold** - `**text**`
  - *Italic* - `*text*`
  - ~~Strikethrough~~ - `~~text~~`
  - ==Mark== - `==text==`
  - `Code` - `` `code` ``
  - הכל בלחיצה אחת 🎨

• **מבנה מסמך**:
  - Headers (H1-H6) - כותרות בכל הגדלים
  - Lists - רשימות ממוספרות ולא ממוספרות
  - Task Lists - `- [ ]` תיבות סימון
  - Blockquotes - ציטוטים
  - Horizontal Rule - קו מפריד 📋

• **תוכן מתקדם**:
  - Links - הוספת קישורים
  - Images - הטמעת תמונות
  - Tables - יצירת טבלאות
  - Code Blocks - בלוקי קוד עם שפה
  - Mermaid Diagrams - תבניות דיאגרמות מוכנות 🖼️

• **קיצורי מקלדת** - עובדים מהמקלדת?
  - `Ctrl/Cmd + B` - Bold
  - `Ctrl/Cmd + I` - Italic
  - `Ctrl/Cmd + K` - Link
  - `Ctrl/Cmd + Shift + C` - Code Block ⌨️

• **הוספה חכמה** - סימנתם טקסט? הכפתור עוטף אותו אוטומטית. לא סימנתם? מוסיף template

• **תבניות מוכנות**:
  - טבלה 3x3
  - דיאגרמת Flowchart
  - Task list עם 3 משימות
  - Code block Python/JavaScript/Bash
  - פשוט קליק ומתחילים למלא 🚀

• **ARIA Labels** - נגישות מלאה לקוראי מסך

• **תמיכה RTL** - עובד מעולה עם עברית
