### פוסט 1: Live Preview עם Split View

```
מכירים את זה שאתם כותבים Markdown ארוך, עוברים לטאב אחר לראות איך זה נראה, מתקנים, חוזרים שוב וחוזר חלילה?

בדיוק בשביל זה הוספתי את Live Preview עם Split View ל-CodeBot.

💻 איך זה עובד?

החלון מתפצל לשניים - משמאל העורך, מימין התצוגה המקדימה. כל תו שאתם מקלידים מתעדכן בזמן אמת בצד השני.

• רינדור Client-Side מהיר ללא שרת
• Debounce חכם של 500ms - לא עומס מיותר
• Hash Deduplication - לא מרנדר פעמיים את אותו תוכן

🎨 מה נתמך?

• Syntax Highlighting עם highlight.js לכל שפה
• דיאגרמות Mermaid מלאות - Flowcharts, Sequence, Gantt, ERD...
• נוסחאות מתמטיות עם KaTeX
• Task Lists עם תיבות סימון אינטראקטיביות
• Admonitions - note, warning, tip, danger
• תוכן עניינים אוטומטי

⌨️ קיצור מקלדת:
Shift+Ctrl+Enter (Windows) או Shift+Cmd+Enter (Mac)

📱 רספונסיבי למובייל עם טאבים במקום split

הפאנלים ניתנים לשינוי גודל (25%-75%), וההעדפות נשמרות ב-localStorage.

נסו את זה עכשיו בחינם:

```

### פוסט 2: Workspace Kanban

```
נמאס לכם לחפש קבצים בין עשרות תיקיות ולא לזכור מה בתהליך ומה הושלם?

בדיוק בשביל זה הוספתי את Workspace Kanban ל-CodeBot.

📋 מה זה בכלל?

לוח Kanban מובנה לניהול קבצי הקוד שלכם. שלוש עמודות פשוטות:

• 📝 לטיפול (To Do) - קבצים שטרם התחלתם
• 🔄 בתהליך (In Progress) - עבודה פעילה
• ✅ הושלם (Done) - סיימתם לטפל

🖱️ Drag & Drop מושלם:

• גרירה בין עמודות - העבירו קבצים בין מצבים בקליק
• גרירה בין אוספים - העבירו קבצים בין אוספים שונים
• SortableJS - גרירה חלקה וטבעית
• עדכון ספירות בזמן אמת

⌨️ קיצורי מקלדת:
• Shift+1 → לטיפול
• Shift+2 → בתהליך  
• Shift+3 → הושלם

🎯 למה זה משנה?

במקום לנהל TODO list נפרד ומערכת קבצים נפרדת - הכל במקום אחד.
הקובץ עצמו הוא המשימה. פשוט וחכם.

התראות נגישות עם aria-live, תמיכה במובייל עם מגע.
