# 💡 רעיונות לפיצ׳רים פרקטיים ל־WebApp (2026)
נסרק: `webapp/static/`, `webapp/static_build/`, `webapp/templates/`, `webapp/*.py`, וגם `webapp/FEATURE_SUGGESTIONS/` כדי להימנע מכפילויות.  
הערה: לא נמצא נתיב `webapp/config/` בפרויקט (יש קבצים כמו `config_radar.py`, אבל לא תיקיית config בתוך `webapp/`).

מיקוד: פיצ׳רים שמועילים למשתמשים בפועל.  
לא כלול: פיצ׳רי Social/Community, ו־AI Agents.

---

### 🏷️ Tag Cleanup Wizard
תיאור: מסך שמרכז את כל התגיות שלך עם ספירת שימושים, ומאפשר איחוד/שינוי שם/ניקוי כפילויות (למשל `#API` מול `#api`) בבת אחת.  
קטגוריה: ארגון וניווט  
מורכבות: בינונית

### 📌 Pin Files to Dashboard
תיאור: אפשרות “נעיצה לדשבורד” לקבצים/אוספים חשובים, כדי להגיע אליהם בקליק בלי לחפש או לזכור פילטרים.  
קטגוריה: דשבורד וזרימת עבודה  
מורכבות: קלה

### 🧭 Tabbed File Navigation (כמו דפדפן)
תיאור: טאבים לקבצים פתוחים + “פתח אחרונים” + שחזור טאבים אוטומטי אחרי רענון/סגירת הדפדפן.  
קטגוריה: ניווט ויעילות  
מורכבות: בינונית

### 🧷 Quick Actions על בחירת טקסט
תיאור: כשמסמנים קטע קוד/Markdown – תפריט קטן שמציע פעולות שימושיות: העתקה, יצירת פתק, יצירת סימנייה/הערה, ושיתוף כקטע (לא כל הקובץ).  
קטגוריה: צפייה ועריכה  
מורכבות: בינונית

### 🧩 “Related Files” Panel (ללא AI)
תיאור: פאנל שמציג “קבצים קשורים” לפי חוקים פשוטים (אותו repo/tag/שפה/תיקייה וירטואלית), עם אפשרות להצמיד ידנית קשרים.  
קטגוריה: ניווט וארגון  
מורכבות: בינונית

### 🗂️ Repo Browser: Breadcrumbs + Jump-to-Path
תיאור: שדרוג חוויית דפדוף ריפו עם פירורי לחם, שדה “קפוץ לנתיב”, והדגשת מיקום הקובץ בעץ.  
קטגוריה: ניווט בריפו  
מורכבות: בינונית

### 🔍 Search “Within Results”
תיאור: אחרי חיפוש גלובלי, אפשרות לסנן את התוצאות שוב בצד לקוח (לפי שפה/תגיות/שם קובץ/גודל/תאריך) בלי עוד roundtrip לשרת.  
קטגוריה: חיפוש  
מורכבות: קלה

### 🧠 Personal Shortcuts (פקודות/טקסטים שמורים)
תיאור: אזור אישי שבו המשתמש שומר “קיצורים” (כמו פקודות CLI, תבניות תגיות, או טקסטים שחוזרים על עצמם) עם כפתור העתקה מהירה.  
קטגוריה: פרודוקטיביות  
מורכבות: קלה

### 🔒 Privacy Screen (Click-to-Reveal)
תיאור: מצב שמסתיר תוכן קוד/Markdown כברירת מחדל (blur/placeholder) ומציג רק אחרי קליק/החזקת מקש, כדי לעבוד ליד אנשים/בשיתוף מסך.  
קטגוריה: פרטיות  
מורכבות: קלה

### 🕵️ Secret Leak Warning לפני שמירה/שיתוף
תיאור: בדיקה מקומית/שרתית מבוססת regex שמתריעה אם הקוד נראה כמו מפתח API/Token/סיסמה, עם אפשרות “המשך בכל זאת” והסבר קצר.  
קטגוריה: אבטחה  
מורכבות: בינונית

### 🧾 “What Changed Since Last Visit”
תיאור: בעמוד צפייה בקובץ, סימון ברור של מה השתנה מאז הפעם האחרונה שפתחת אותו (על בסיס גרסאות/updated_at) עם קפיצה לשינויים.  
קטגוריה: צפייה והבנה  
מורכבות: בינונית

### 🧺 Bulk Move to Trash by Rule (Preview + Confirm)
תיאור: מסך שמאפשר “ניקוי חכם” לפי תנאים פשוטים (לא נפתח X ימים / גדול מ־Y / תגית מסוימת) עם Preview לפני ביצוע.  
קטגוריה: ניהול קבצים  
מורכבות: בינונית

### 📦 Project Bundle Export (Structure-Preserving)
תיאור: ייצוא ZIP שמכבד מבנה תיקיות “וירטואלי”/נתיבי ריפו, ומוסיף README אוטומטי שמרכז מטא־דאטה (שמות קבצים, תגיות, תיאורים).  
קטגוריה: ייצוא וגיבוי  
מורכבות: בינונית

### 🧮 Smart Diff for JSON/YAML
תיאור: בהשוואה בין גרסאות/קבצים מסוג JSON/YAML – תצוגת diff שמבינה מבנה (מפתחות/מערכים) ומפחיתה “רעש” של פורמט/רווחים.  
קטגוריה: השוואה (Compare)  
מורכבות: מורכבת

### 🧷 Side-by-Side: Code + Rendered Markdown (בעת עריכה)
תיאור: בעת עריכת Markdown, תצוגה מפוצלת: משמאל עורך, מימין Preview חי, עם גלילה מסונכרנת וקפיצה לכותרות.  
קטגוריה: עריכת Markdown  
מורכבות: מורכבת

### 🧰 “Scratchpad” זמני בתוך ה־WebApp
תיאור: דף טיוטה מהיר (Markdown/טקסט) לשמירת רעיונות/הדבקות זמניות, עם שמירה מקומית אוטומטית וכפתור “הפוך לקובץ”.  
קטגוריה: פרודוקטיביות  
מורכבות: קלה

### 🗺️ Global “Jump Map” לפי ישויות
תיאור: מסך שמרכז “ישויות” נפוצות (תגיות, ריפואים, אוספים, שפות) עם חיפוש מהיר וקיצורי מקלדת כדי לקפוץ לתצוגה הנכונה.  
קטגוריה: ניווט  
מורכבות: בינונית

### 🧷 One-Click Copy as Snippet (עם כותרת ותיאור)
תיאור: מהצפייה בקובץ: לבחור טווח, לתת שם קצר, ולשמור אותו כפריט חדש (סניפט/קובץ) עם תיוג אוטומטי והפניה למקור.  
קטגוריה: שימוש חוזר  
מורכבות: בינונית

### ♿ Accessibility Audit Mode
תיאור: “מצב בדיקה” שמדגיש בעיות A11y נפוצות ב־UI (פוקוס, aria-label חסר, ניגודיות), עם רשימת תיקונים מומלצים למשתמש/מפתח.  
קטגוריה: נגישות  
מורכבות: מורכבת
