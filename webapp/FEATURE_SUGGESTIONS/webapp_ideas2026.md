# 💡 WebApp Feature Suggestions 2026

**תאריך:** ינואר 2026
**מיקוד:** פיצ'רים שימושיים למשתמשים, כלים למפתחים, ושיפורי UX.
**הוחרגו:** Social/Community, AI Agents.

---

### 🛠️ JSON to Typescript/Pydantic Converter
**תיאור:** כלי להמרת JSON ל-Type Definitions (TS/Python/Go) ישירות מהדפדפן. כולל זיהוי סוגים אוטומטי ואפשרות להעתקה מהירה.
**קטגוריה:** Developer Tools
**מורכבות:** בינונית

### 🔍 Regex Search Builder
**תיאור:** ממשק ויזואלי לבניית ביטויים רגולריים לחיפוש בקוד. כולל הסברים בזמן אמת על כל חלק בביטוי ובדיקה על טקסט לדוגמה.
**קטגוריה:** Search
**מורכבות:** בינונית

### 📖 Bionic Reading Mode
**תיאור:** מצב קריאה המדגיש את האותיות הראשונות בכל מילה (Bold) לשיפור מהירות הקריאה והמיקוד בקבצי טקסט ארוכים.
**קטגוריה:** Accessibility/UX
**מורכבות:** קלה

### 🧩 Dashboard Layout Customization
**תיאור:** אפשרות לגרירה (Drag & Drop) ושינוי סדר הווידג'טים בדשבורד הראשי, כולל הסתרה/הצגה של כרטיסים לפי צורך אישי.
**קטגוריה:** UX
**מורכבות:** בינונית

### 📸 Snippet "Copy as Image"
**תיאור:** יצירת תמונה מעוצבת (בסגנון Carbon) מקטע קוד נבחר, עם מסגרת, הצללה ורקע, לשיתוף ברשתות חברתיות או מסמכים.
**קטגוריה:** Developer Tools
**מורכבות:** בינונית

### 🎨 Theme Export/Import
**תיאור:** אפשרות לייצא את ערכת הנושא הנוכחית (CSS Variables) לקובץ JSON ולייבא ערכות משתמשים אחרים או לגיבוי.
**קטגוריה:** Customization
**מורכבות:** קלה

### 🖼️ Image Metadata Viewer
**תיאור:** חלונית מידע בצפייה בתמונות המציגה נתוני EXIF (מצלמה, מיקום, תאריך) ומידע טכני (גודל, רזולוציה, פורמט).
**קטגוריה:** Tools
**מורכבות:** קלה

### 🕒 Local Time Conversion
**תיאור:** המרה אוטומטית של כל חותמות הזמן (UTC) בממשק לשעון המקומי של הדפדפן, עם Tooltip המציג את הזמן המקורי.
**קטגוריה:** UX
**מורכבות:** קלה

### 📊 CSV Viewer/Editor
**תיאור:** תצוגת טבלה אינטראקטיבית לקבצי CSV במקום טקסט גולמי, עם אפשרויות מיון עמודות, סינון, וחיפוש בתוך הטבלה.
**קטגוריה:** Tools
**מורכבות:** בינונית

### 🎨 Color Picker from Image
**תיאור:** כלי המאפשר לדגום צבעים מתוך תמונות שהועלו למערכת ולקבל את קוד הצבע (HEX/RGB) ואף לייצר פלטת צבעים מהתמונה.
**קטגוריה:** Tools
**מורכבות:** בינונית

### ⚡ SVG Preview & Optimizer
**תיאור:** תצוגה מקדימה ויזואלית לקבצי SVG (במקום קוד) עם כפתור לאופטימיזציה (הקטנת משקל הקובץ) וניקוי קוד מיותר.
**קטגוריה:** Tools
**מורכבות:** בינונית

### 🗺️ Minimap for Long Files
**תיאור:** תצוגה ממוזערת של הקובץ בצד המסך (כמו ב-VS Code) המאפשרת התמצאות וגלילה מהירה בקבצי קוד ארוכים.
**קטגוריה:** UX
**מורכבות:** בינונית

### 🧜‍♀️ Mermaid Diagram Renderer
**תיאור:** זיהוי ורינדור אוטומטי של בלוקי קוד מסוג `mermaid` בתוך קבצי Markdown לדיאגרמות ויזואליות חיות.

**מורכבות:** בינונית

**תיאור:** קבלת התראות דפדפן (Native Notifications) על סיום ג'ובים ארוכים או שגיאות קריטיות, גם כשהטאב אינו בפוקוס.

**מורכבות:** קלה
