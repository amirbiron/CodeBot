## 📍 ניווט וארגון קבצים

### 📌 File Pinning - הצמדת קבצים לדשבורד
**תיאור:** אפשרות להצמיד קבצים חשובים לראש הדשבורד עם גישה מהירה בקליק אחד.  
**קטגוריה:** Navigation  
**מורכבות:** קל

---

### 🗂️ Tab-Based File Browser
**תיאור:** פתיחת מספר קבצים במקביל בלשוניות (כמו IDE), עם אפשרות להחליף ביניהם בלי לאבד מיקום.  
**קטגוריה:** Navigation  
**מורכבות:** בינוני

---

### 📊 File Usage Analytics
**תיאור:** סטטיסטיקות צפיות ועריכות לכל קובץ - כמה פעמים נפתח, מתי נערך לאחרונה, ממי (במצב מרובה מכשירים).  
**קטגוריה:** Analytics  
**מורכבות:** בינוני

---

### 🔗 Quick File Duplicate
**תיאור:** כפתור לשכפול מהיר של קובץ קיים עם שם חדש, כולל אפשרות לערוך מיד אחרי השכפול.  
**קטגוריה:** File Management  
**מורכבות:** קל

---

### 📂 Nested Collections (תיקיות משנה)
**תיאור:** אפשרות ליצור אוספים בתוך אוספים (היררכיה עד 3 רמות) לארגון מתקדם יותר.  
**קטגוריה:** Organization  
**מורכבות:** בינוני

---

## ✏️ שיפורי עריכה

### 📏 Split View Editor
**תיאור:** פיצול מסך העריכה לשניים - לצפייה או השוואה בין שני חלקים באותו קובץ, או בין שני קבצים שונים.  
**קטגוריה:** Editor  
**מורכבות:** מורכב

---

### 🎨 Personal Syntax Theme Editor
**תיאור:** עורך ויזואלי לצבעי syntax highlighting אישיים - בחירת צבע לכל סוג token (keyword, string, comment) עם תצוגה מקדימה חיה.  
**קטגוריה:** Customization  
**מורכבות:** בינוני

---

### 📝 Inline Code Annotations
**תיאור:** הוספת הערות קצרות (annotations) לשורות ספציפיות בקוד שנשמרות בנפרד מהקובץ - שימושי לביקורת עצמית ולמידה.  
**קטגוריה:** Editor  
**מורכבות:** בינוני

---

### 🔢 Line Bookmarks
**תיאור:** סימון שורות חשובות בתוך קובץ (bookmark פנימי) עם קפיצה מהירה אליהן - שימושי לקבצים ארוכים.  
**קטגוריה:** Navigation  
**מורכבות:** קל

---

### ⌨️ Code Snippets Expansion
**תיאור:** הגדרת קיצורים אישיים שמתרחבים לקוד מלא (כמו `fn→` מתרחב לפונקציה שלמה בשפה הנבחרת).  
**קטגוריה:** Editor  
**מורכבות:** בינוני

---

### 📋 Clipboard History
**תיאור:** היסטוריית הלוח האחרונה (5-10 פריטים) בתוך העורך - גישה מהירה לקטעי קוד שהועתקו לאחרונה.  
**קטגוריה:** Editor  
**מורכבות:** קל

---

## 🔍 חיפוש וגילוי

### 🔎 Semantic Search (לפי משמעות)
**תיאור:** חיפוש קוד לפי מה שהוא עושה (למשל: "loop over array") ולא רק לפי טקסט מדויק - שימוש ב-embeddings פשוטים.  
**קטגוריה:** Search  
**מורכבות:** מורכב

---

### 📈 Code Complexity Meter
**תיאור:** מד מורכבות פשוט לכל קובץ - מספר שורות, עומק קינון מקסימלי, מספר פונקציות - עם התראה על קבצים "כבדים".  
**קטגוריה:** Analytics  
**מורכבות:** בינוני

---

### 🔄 Find & Replace Across Files
**תיאור:** חיפוש והחלפה גלובלית בכל הקבצים (או בתיקייה/אוסף מסוים) עם preview לפני ביצוע.  
**קטגוריה:** Search  
**מורכבות:** בינוני

---

### 🏷️ Smart Tag Suggestions
**תיאור:** הצעות אוטומטיות לתגיות על בסיס תוכן הקובץ (זיהוי frameworks, patterns, שפות).  
**קטגוריה:** Organization  
**מורכבות:** בינוני

## 🛠️ כלים ותשתית

### 💾 Local Auto-Backup
**תיאור:** שמירה אוטומטית של טיוטות ב-localStorage כל X שניות - שחזור במקרה של סגירת הדפדפן בטעות.  
**קטגוריה:** Reliability  
**מורכבות:** קל

---

### 📤 Quick Share to Clipboard
**תיאור:** כפתור אחד שמייצר קישור שיתוף זמני (1 שעה) ומעתיק אותו ללוח מיידית - בלי פתיחת דיאלוגים.  
**קטגוריה:** Sharing  
**מורכבות:** קל

---

### 🔐 Environment Variables Vault
**תיאור:** אחסון מאובטח למשתני סביבה (.env) עם הצפנה - לשמירת API keys וסודות בנפרד מהקוד.  
**קטגוריה:** Security  
**מורכבות:** מורכב

---

### 📊 Code Statistics Dashboard
**תיאור:** דשבורד מרכזי עם סטטיסטיקות על כל הקבצים - חלוקה לפי שפה, גודל ממוצע, תדירות עדכון, trends לאורך זמן.  
**קטגוריה:** Analytics  
**מורכבות:** בינוני

---

### ⏰ Read Time Estimator
**תיאור:** הערכת זמן קריאה לכל קובץ (בהתאם לשפה ומורכבות) - מוצג בכרטיס הקובץ.  
**קטגוריה:** UX  
**מורכבות:** קל
