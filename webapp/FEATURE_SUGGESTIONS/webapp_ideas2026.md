## ⌨️ עריכה ופרודוקטיביות

### 🔧 Keyboard Shortcuts Manager
**תיאור:** ממשק ויזואלי להגדרת קיצורי מקלדת אישיים - המשתמש יכול לשנות את ה-bindings לכל פעולה נפוצה.  
**קטגוריה:** Productivity  
**מורכבות:** בינונית

---

### 📝 Smart Snippets Library (Personal)
**תיאור:** ספריית קטעי קוד אישית עם placeholders (כמו `${1:name}`) - הקלד trigger word וקבל expansion אוטומטי.  
**קטגוריה:** Editor  
**מורכבות:** בינונית

---

### ✂️ Split View Editor
**תיאור:** פיצול העורך לשני חלונות זה-לצד-זה (או אנכית) לעריכת/השוואת שני קבצים במקביל.  
**קטגוריה:** Editor  
**מורכבות:** מורכב

---

### 💾 Auto-Save Drafts (Local)
**תיאור:** שמירה אוטומטית מקומית ל-localStorage כל 30 שניות בזמן עריכה, עם התראה אם יש טיוטה שלא נשמרה.  
**קטגוריה:** Editor  
**מורכבות:** קלה

---

### 🔄 Cross-File Search & Replace
**תיאור:** חיפוש והחלפה בכל הקבצים (או בבחירה) בפעולה אחת, עם preview לפני ביצוע ואפשרות undo.  
**קטגוריה:** Editor  
**מורכבות:** מורכב

---

### 📋 Clipboard History Panel
**תיאור:** פאנל צדדי שמציג את 20 הפריטים האחרונים שהועתקו - לחיצה מדביקה לעורך.  
**קטגוריה:** Editor  
**מורכבות:** בינונית

---

### 🧱 Code Block Folding UI
**תיאור:** כפתורים ויזואליים לקיפול/פריסה של בלוקים לוגיים (פונקציות, מחלקות, לולאות) בצד שמאל של מספרי השורות.  
**קטגוריה:** Editor  
**מורכבות:** בינונית

---

### 📏 Visual Indentation Guides
**תיאור:** קווים אנכיים דקים שמציינים רמות הזחה - מקל על קריאת Python/YAML ושפות מבוססות רווחים.  
**קטגוריה:** Editor  
**מורכבות:** קלה

---

## 🔍 ניווט וחיפוש

### 👁️ File Preview on Hover
**תיאור:** ב-hover מעל שם קובץ ברשימה - popup קטן עם 10-15 השורות הראשונות (או תיאור) ללא צורך בקליק.  
**קטגוריה:** Navigation  
**מורכבות:** בינונית

---

### ⚡ Quick Actions Radial Menu
**תיאור:** לחיצה ימנית על קובץ פותחת תפריט רדיאלי (עיגולי) עם פעולות נפוצות: עריכה, מועדפים, שיתוף, מחיקה.  
**קטגוריה:** Navigation  
**מורכבות:** בינונית

---

### 🧭 Breadcrumb Navigation Enhanced
**תיאור:** שדרוג שביל הפירורים: כל רכיב לחיץ עם dropdown שמציג "אחים" באותה רמה (קבצים באותו פרויקט/תגית).  
**קטגוריה:** Navigation  
**מורכבות:** בינונית

---

### 🔖 Go to Line/Symbol Dialog
**תיאור:** קיצור `Ctrl+G` פותח דיאלוג: הקלד מספר שורה לקפיצה, או הקלד שם פונקציה/מחלקה (symbol) לניווט מהיר.  
**קטגוריה:** Navigation  
**מורכבות:** בינונית

---

### 🕐 Recent Files Sidebar
**תיאור:** פאנל צדדי קבוע שמציג 10-15 קבצים אחרונים שנצפו/נערכו, עם מעבר בקליק אחד.  
**קטגוריה:** Navigation  
**מורכבות:** קלה

---

### 🔗 Smart Internal Links
**תיאור:** בתיאור קובץ, זיהוי אוטומטי של `@filename` והפיכתו לקישור פנימי לקובץ אחר במערכת.  
**קטגוריה:** Navigation  
**מורכבות:** קלה

---

## 📊 ויזואליזציה ותצוגה

### 🎯 Code Complexity Score Badge
**תיאור:** תג ויזואלי קטן בכרטיס הקובץ שמציג ציון מורכבות (פשוט/בינוני/מורכב) על בסיס שורות, עומק קינון ומספר פונקציות.  
**קטגוריה:** Visualization  
**מורכבות:** בינונית

---

### 📚 Reading Time Estimator
**תיאור:** הערכת זמן קריאה (למשל: "~3 דק' קריאה") המוצגת בכותרת דף הצפייה - מבוססת על מספר שורות ומורכבות.  
**קטגוריה:** Visualization  
**מורכבות:** קלה

---

### 🖨️ Print-Optimized Mode
**תיאור:** כפתור "הכן להדפסה" שמייצר גרסה מותאמת: פונט מודפס, רקע לבן, מספור שורות, כותרת עם metadata.  
**קטגוריה:** Export  
**מורכבות:** קלה

---

### 🌈 Syntax Theme Quick Switcher
**תיאור:** כפתור קטן בעורך שמאפשר החלפת theme הדגשת תחביר "בטיסה" (ללא כניסה להגדרות) - preview מיידי.  
**קטגוריה:** Visualization  
**מורכבות:** קלה

---

### 📐 Line Length Indicator
**תיאור:** קו אנכי דק במיקום 80/120 תווים להצגת "גבול מומלץ" לאורך שורה (PEP8/style guides).  
**קטגוריה:** Editor  
**מורכבות:** קלה

---

### 🔢 Line Numbers Enhancement
**תיאור:** שיפור מספור שורות: hover מציג מספר יחסי (מהשורה הנוכחית), קליק מסמן את השורה, double-click מעתיק את השורה.  
**קטגוריה:** Editor  
**מורכבות:** קלה

---

## 📦 ייבוא וייצוא

### 🌐 Import from URL
**תיאור:** הדבק URL של קובץ קוד (GitHub raw, Gist, Pastebin) - המערכת תשלוף ותשמור אוטומטית.  
**קטגוריה:** Import  
**מורכבות:** בינונית

---

### 📋 Paste & Auto-Detect
**תיאור:** העתק קוד מכל מקום, הדבק בשדה ייעודי - זיהוי אוטומטי של שפה, הצעת שם קובץ ושמירה מהירה.  
**קטגוריה:** Import  
**מורכבות:** קלה

---

### 📤 Export Presets Manager
**תיאור:** שמירת הגדרות ייצוא מועדפות (פורמט, theme, metadata) כ-presets - בחירה מהירה בייצוא הבא.  
**קטגוריה:** Export  
**מורכבות:** קלה

---

### 📧 Email Code Snippet
**תיאור:** כפתור "שלח במייל" שפותח mailto עם הקוד מפורמט (או לינק לשיתוף) - לשיתוף מהיר עם עמיתים.  
**קטגוריה:** Export  
**מורכבות:** קלה

---

### 📱 QR Code for File
**תיאור:** יצירת QR code לקובץ משותף - סריקה במובייל מעבירה ישירות לדף הצפייה.  
**קטגוריה:** Export  
**מורכבות:** קלה

---

## 🗂️ ארגון וניהול קבצים

### 🔍 Duplicate Detector
**תיאור:** סריקה אוטומטית לזיהוי קבצים דומים/כפולים (על בסיס hash או fuzzy matching) - הצעה למיזוג או מחיקה.  
**קטגוריה:** Organization  
**מורכבות:** בינונית

---

### 🏷️ Auto-Tagging Suggestions
**תיאור:** בשמירת קובץ, הצעת תגיות אוטומטית על בסיס: שפה, imports, שמות פונקציות, דפוסים נפוצים.  
**קטגוריה:** Organization  
**מורכבות:** בינונית

---

### 📁 Smart Folders (Virtual)
**תיאור:** יצירת תיקיות וירטואליות על בסיס חוקים (למשל: "כל קבצי Python מ-2025") - דינמי, לא מעביר קבצים.  
**קטגוריה:** Organization  
**מורכבות:** מורכב

---

### 🎨 Color Labels for Files
**תיאור:** הוספת צבע (אדום/צהוב/ירוק/כחול) לקבצים - סימון מהיר של סטטוס או עדיפות.  
**קטגוריה:** Organization  
**מורכבות:** קלה

---

### 📌 Pin Files to Top
**תיאור:** הצמדת קבצים חשובים לראש הרשימה - תמיד זמינים ללא חיפוש.  
**קטגוריה:** Organization  
**מורכבות:** קלה

---

### 🗃️ Archive Mode
**תיאור:** העברת קבצים ישנים ל"ארכיון" - לא נמחקים, לא מופיעים ברשימה הראשית, נגישים דרך מסנן.  
**קטגוריה:** Organization  
**מורכבות:** קלה

## 📈 תובנות ומעקב

### 📆 Session History Timeline
**תיאור:** ציר זמן ויזואלי של הפעולות של היום: קבצים שנוצרו/נערכו/נצפו, עם אפשרות לחזור לכל נקודה.  
**קטגוריה:** Analytics  
**מורכבות:** בינונית

---

### 📊 Language Distribution Chart
**תיאור:** גרף עוגה/עמודות המציג התפלגות שפות התכנות בקבצים שלך - תובנה על מה אתה עובד.  
**קטגוריה:** Analytics  
**מורכבות:** קלה

---

### 🏆 Weekly Activity Summary
**תיאור:** סיכום שבועי אוטומטי: כמה קבצים נוספו, נערכו, מה השפה הכי פעילה, מה הקובץ הכי נצפה.  
**קטגוריה:** Analytics  
**מורכבות:** קלה

---

### 🔔 File Update Notifications
**תיאור:** הגדרת "מעקב" על קובץ - התראה (push/email) כשהוא מתעדכן (רלוונטי לקבצים משותפים עתידיים).  
**קטגוריה:** Analytics  
**מורכבות:** בינונית

---

### 📉 Storage Usage Dashboard
**תיאור:** דשבורד קטן המציג: כמה מקום נוצל, כמה קבצים, ממוצע גודל קובץ, טרנד לאורך זמן.  
**קטגוריה:** Analytics  
**מורכבות:** קלה

## ♿ נגישות וחוויית משתמש

### 🎹 Focus Mode (Distraction-Free)
**תיאור:** מצב עריכה נקי: הסתרת sidebar, navbar, metadata - רק העורך במרכז המסך עם רקע רגוע.  
**קטגוריה:** UX  
**מורכבות:** קלה

---

### 📖 Reading View Mode
**תיאור:** מצב קריאה אופטימלי: פונט גדול יותר, רווח שורות מוגדל, רוחב מוגבל (60-80 תווים) - נוח לקריאת קוד.  
**קטגוריה:** UX  
**מורכבות:** קלה

---

### 🔤 Font Size Slider
**תיאור:** סליידר בזמן אמת לשינוי גודל פונט בעורך (12px-24px) - ללא כניסה להגדרות.  
**קטגוריה:** Accessibility  
**מורכבות:** קלה

---

### 🖱️ Scroll Progress Indicator
**תיאור:** פס דק בצד ימין שמראה איפה אתה בקובץ (כמו minimap מינימלי) - שימושי לקבצים ארוכים.  
**קטגוריה:** UX  
**מורכבות:** קלה

---

### ⏪ Undo/Redo History Panel
**תיאור:** פאנל שמציג היסטוריית שינויים (לא רק Ctrl+Z) - לחיצה על פריט מחזירה לאותה נקודה.  
**קטגוריה:** Editor  
**מורכבות:** בינונית

---

### 🌙 Auto Dark Mode
**תיאור:** זיהוי אוטומטי של העדפת המערכת (prefers-color-scheme) והתאמת ה-theme בהתאם.  
**קטגוריה:** UX  
**מורכבות:** קלה

---

### 💬 Inline Notes/Comments
**תיאור:** הוספת הערות אישיות (לא בקוד עצמו) על שורות ספציפיות - מוצגות כ-tooltip, נשמרות בנפרד.  
**קטגוריה:** Editor  
**מורכבות:** בינונית
