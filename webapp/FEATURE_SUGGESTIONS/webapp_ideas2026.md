# 💡 WebApp Feature Suggestions 2026

מסמך זה מרכז רעיונות לפיצ'רים חדשים עבור ה-WebApp, עם דגש על שיפור חווית המשתמש וכלים פרקטיים למפתחים.

## 🛠️ Developer Tools

### 🧪 Regex Builder Assistant
תיאור: חלונית צד לבנייה ובדיקה של ביטויים רגולריים (Regex) בזמן אמת מול הטקסט בעורך, כולל הסברים ויזואליים על התבנית.
קטגוריה: Developer Tools
מורכבות: בינונית

### 🎨 Color Palette Extractor
תיאור: סריקה אוטומטית של קודי צבע (Hex, RGB, HSL) בקובץ והצגתם בפאנל צדדי המאפשר עריכה ושינוי גוונים בקלות.
קטגוריה: Design Tools
מורכבות: בינונית

### 📝 Dummy Data Generator
תיאור: כלי להוספה מהירה של נתוני דמה (Lorem Ipsum, JSON, CSV, שמות, כתובות) ישירות לתוך העורך לצורכי בדיקה.
קטגוריה: Developer Tools
מורכבות: קלה

## 📊 Visualization & Analysis

### 🕸️ Import Dependency Graph
תיאור: ויזואליזציה גרפית של הקשרים והתלויות (Imports/Requires) של הקובץ הנוכחי עם קבצים אחרים בפרויקט.
קטגוריה: Visualization
מורכבות: מורכב

### 🌡️ Code Complexity Heatmap
תיאור: אינדיקטור ויזואלי (ב-Gutter או ב-Scrollbar) המציג אזורים מורכבים בקוד על בסיס עומק ההזחה (Indentation Depth).
קטגוריה: Code Quality
מורכבות: בינונית

## 🚀 Productivity & Editor UX

### 📽️ Markdown Presentation Mode
תיאור: מצב תצוגה ההופך קובץ Markdown למצגת שקפים מעוצבת (בסגנון Reveal.js) בלחיצת כפתור.
קטגוריה: Productivity
מורכבות: בינונית

### ⌨️ Visual Keybinding Mapper
תיאור: ממשק גרפי לשינוי והתאמה אישית של קיצורי המקלדת (Keyboard Shortcuts) לכל הפעולות במערכת.
קטגוריה: Customization
מורכבות: בינונית

### 🕰️ Local Session Time Machine
תיאור: סליידר המאפשר "חזרה בזמן" לכל השינויים שבוצעו ב-Session הנוכחי (Local History), גם לפני השמירה לשרת.
קטגוריה: Safety / Editor
מורכבות: בינונית

### 🎭 Typewriter Scrolling Mode
תיאור: מצב עריכה השומר תמיד את השורה הפעילה במרכז המסך אנכית, כדי למנוע מאמץ בצוואר ובעיניים.
קטגוריה: UX
מורכבות: קלה

### 📋 Clipboard Diff Viewer
תיאור: כלי להשוואה מהירה (Diff) בין הטקסט המסומן בעורך לבין התוכן הנוכחי בלוח הגזירים (Clipboard).
קטגוריה: Utilities
מורכבות: קלה
