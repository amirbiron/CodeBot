# 💡 WebApp Feature Suggestions (User-First) — 2026
מיקוד: פיצ׳רים שימושיים שקשה לסרב להם (פרודוקטיביות, ניהול ידע, חיפוש, שיתוף בטוח, UX).  
לא כלול: פיצ׳רי קהילה/סושיאל, AI Agents.  

> הערה: בתיקייה הזו כבר קיימים מסמכים נוספים (למשל `webapp_ideas3.md`, `webapp_improvement_ideas.md`).  
> הרשימה כאן נבנתה אחרי סריקה של `webapp/templates/`, `webapp/static/`, `webapp/static_build/`, `webapp/config*` ובמכוון מנסה **לא לשכפל** רעיונות שכבר כתובים שם.

---

### 🧾 מרכז טיוטות (Draft Recovery Center)
תיאור: מסך שמציג את כל הקבצים שיש להם טיוטה לא־שמורה (כולל “מתי עודכן לאחרונה”) עם כפתורי שחזור/מחיקה.  
קטגוריה: פרודוקטיביות / עריכה  
מורכבות: בינונית

### 🖼️ הדבקת תמונה ל‑Markdown (Paste → Upload → Insert)
תיאור: הדבקת צילום מסך (Ctrl+V) בעורך תעלה את התמונה ותכניס אוטומטית Markdown link במקום הסמן.  
קטגוריה: Markdown / UX  
מורכבות: בינונית

### 🔐 מצב “שיתוף בטוח” (Redaction לפני שיתוף)
תיאור: בעת יצירת קישור שיתוף, אפשרות לטשטוש אוטומטי של סודות נפוצים (API keys, tokens, .env) עם כפתור “חשוף זמנית”.  
קטגוריה: אבטחה / שיתוף  
מורכבות: בינונית

### 🔗 מרכז קישורי שיתוף (ניהול, ביטול, תפוגה)
תיאור: עמוד אחד שמציג את כל קישורי השיתוף שיצרת (לקובץ/ל‑bulk), עם ביטול קישור, שינוי תפוגה, והעתקה מהירה.  
קטגוריה: שיתוף / ניהול  
מורכבות: בינונית

### 🧷 “שמור כתצוגה” → אוספים חכמים
תיאור: להפוך חיפוש/פילטרים (שפה/תגיות/ריפו/גדול/אחרון) ל”אוסף חכם” שמתעדכן לבד ומופיע בסיידבר/דשבורד.  
קטגוריה: ארגון / ניווט  
מורכבות: בינונית

### 🧹 מנהל תגיות (איחוד/שינוי שם/ניקוי)
תיאור: מסך שמרכז את כל התגיות שלך, כולל “איחוד תגיות כפולות”, שינוי שם, ומחיקה בטוחה עם תצוגת “כמה קבצים יושפעו”.  
קטגוריה: ארגון  
מורכבות: בינונית

### 🧭 חיפוש “מוגבל להקשר” (ריפו/אוסף/מועדפים)
תיאור: אותו חיפוש גלובלי, אבל עם Toggle ברור שמגביל את התוצאות להקשר הנוכחי (ריפו נבחר / אוסף / מועדפים).  
קטגוריה: חיפוש  
מורכבות: קלה

### 🧩 טאבים/Stack לקבצים שנפתחו לאחרונה
תיאור: פס קטן שמציג 5–10 קבצים שנפתחו לאחרונה עם “חזרה” מהירה בלי לאבד פילטרים/עמוד ברשימה.  
קטגוריה: ניווט / UX  
מורכבות: בינונית

### 🧯 “מצב חירום” למחיקות (Undo מהיר מסל)
תיאור: אחרי העברה לסל (יחיד/קבוצתי) להציג Toast עם כפתור “בטל” ל־10 שניות, כדי להציל טעויות.  
קטגוריה: UX / בטיחות  
מורכבות: קלה

### 🧠 דוח “איפה הזמן שלי הולך” (סטטיסטיקות שימוש אישיות)
תיאור: דשבורד אישי: אילו תגיות/ריפואים/שפות אתה פותח הכי הרבה, קבצים “חמים” בשבוע האחרון, וקבצים שלא נגעת בהם זמן רב.  
קטגוריה: תובנות / ניהול  
מורכבות: בינונית

### 🧪 “תצוגת בדיקות מהירה” לקבצי קונפיג
תיאור: עבור JSON/YAML/ENV: כפתור בדיקה שמריץ ולידציה בסיסית ומדגיש שורה/מפתח בעייתי (לפחות בצד לקוח איפה שאפשר).  
קטגוריה: כלים / קונפיג  
מורכבות: בינונית

### 🧰 “הכנס קטע” (Snippet Picker אישי)
תיאור: חלונית שמאפשרת לשמור קטעים חוזרים (boilerplate) ולהכניס אותם לעורך עם חיפוש קצר.  
קטגוריה: פרודוקטיביות / עריכה  
מורכבות: בינונית

### 🧱 קיפול מקטעים בתצוגת צפייה (Fold בצד צפייה)
תיאור: בתצוגת קובץ (לא עריכה), אפשרות לקפל blocks (למשל functions/classes/markdown headings) כדי לקרוא מהר קבצים ארוכים.  
קטגוריה: קריאה / ניווט  
מורכבות: מורכבת

### 🎚️ העדפות תצוגה לכל קובץ (Wrap/Font/View Mode)
תיאור: שמירת העדפות פר‑קובץ כמו עטיפת שורות, גודל פונט, ותצוגה מתקדמת/בסיסית—כדי שלא “נלחמים” כל פעם מחדש.  
קטגוריה: UX  
מורכבות: בינונית

### 📌 לוח “נעוצים” חכם בדשבורד
תיאור: אפשרות לסדר נעוצים בגרירה, לקבץ לפי ריפו/תגית, ולהוסיף “כותרות” (sections) כדי להפוך את הדשבורד לשולחן עבודה אמיתי.  
קטגוריה: ארגון / דשבורד  
מורכבות: בינונית

### 🗃️ מרכז ייצוא (Export Hub)
תיאור: מסך שמציג את כל הייצואים האחרונים (ZIP/HTML מעוצב וכו׳), עם הורדה מחדש, מחיקה, ותווית “מאיזה ריפו/אוסף זה נוצר”.  
קטגוריה: ייצוא / ניהול  
מורכבות: בינונית

### 🧷 “העתק כ‑Patch” מהשוואה/היסטוריה
תיאור: כפתור שמייצר Patch קטן (unified diff) בין שתי גרסאות כדי להדביק ב‑PR/צ׳אט במהירות.  
קטגוריה: השוואה / פרודוקטיביות  
מורכבות: בינונית

### 🧵 “הערות אישיות” לקובץ (Private Notes)
תיאור: שדה הערות פנימי (לא בתוכן הקובץ) לדברים כמו “למה עשיתי את זה”, “מה לבדוק”, “איפה זה בשירות”.  
קטגוריה: ארגון / תיעוד  
מורכבות: בינונית

### 🧭 סייר ריפו מתקדם (Tree + Quick Jump)
תיאור: כשנכנסים לקטגוריית ריפו: להציג עץ תיקיות ו‑Quick Jump לקבצים לפי path, כדי לעבוד כמו “מיני‑IDE”.  
קטגוריה: ניווט / ריפו  
מורכבות: מורכבת

### 🧮 “ניקוי כפילויות” (Duplicate Finder) עם UI
תיאור: תצוגה שמרכזת קבצים שנראים זהים/מאוד דומים (שם/גודל/חתימה), עם פעולות: מיזוג, שמירה אחת, או תיוג “duplicate”.  
קטגוריה: ארגון / ניקיון  
מורכבות: מורכבת

### 🧷 קיצורי “העתקה חכמה” (Copy as…)
תיאור: ליד “העתק קוד” להוסיף בחירה: העתק בלי מספרי שורות, העתק עם שם קובץ וכותרת, או העתק כ‑Markdown code block.  
קטגוריה: UX / שיתוף  
מורכבות: קלה

### 📎 Drag & Drop משופר להעלאה
תיאור: גרירת קבצים לכל מסך (Files/Upload) תפתח “מגירת העלאה” עם תור העלאות, התקדמות, ושיוך אוטומטי לריפו אם זוהה.  
קטגוריה: העלאה / UX  
מורכבות: בינונית

### 🧾 “רשימת משימות מתוך תגיות” (TODO Dashboard בלי AI)
תיאור: דשבורד שמציג קבצים עם תגית `todo`/`fixme` + חיפוש מהיר בתוך הקוד למילות מפתח, כדי לנהל עבודה בלי לצאת החוצה.  
קטגוריה: פרודוקטיביות  
מורכבות: בינונית

### 🛡️ נעילת מסך לקבצים רגישים (Protect View)
תיאור: אפשרות לסמן קובץ כ“רגיש” כך שהצפייה/שיתוף דורשים אישור נוסף (PIN קצר/לחיצה כפולה) ומסתירים תצוגה מקדימה ברשימות.  
קטגוריה: אבטחה / פרטיות  
מורכבות: מורכבת

### 🧭 “מרכז קיצורים” בעורך (Help Overlay)
תיאור: חלונית עזרה קטנה שמציגה קיצורים פעילים לפי העורך שנבחר (CodeMirror/textarea) כולל חיפוש “מה הקיצור ל…”.  
קטגוריה: UX / עריכה  
מורכבות: קלה
