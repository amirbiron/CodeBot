### High Contrast
- מקור: `webapp/static/css/high-contrast.css` + overrides ב-`variables.css`
- רקע כללי: `#000000`
- טקסטים וקישורים: גוף `#ffffff`, קישורים `#ffff00` (hover `#ffcc00`)
- כפתורים, קלטים וכרטיסים: רקע שחור עם גבול `#ffffff`; `--text-on-primary` ו-`--text-on-warning` מוגדרים ל-`#000000` כדי להבטיח ניגודיות
- פוקוס: outline צהוב `#ffff00`
- Glass: שחור עם גבול לבן (`--glass-border: #ffffff`)
- קוד (CodeMirror/Codemirror):
  - keyword `#ff00ff`
  - string `#00ff00`
  - comment `#00ffff`
  - number `#ffff00`
  - error רקע `#ff0000` וטקסט שחור
- בחירה (selection): רקע `#ffff00`, טקסט שחור

## הערות נוספות

- ערכות Classic/Ocean/Forest יורשות את צבעי `--success`, `--danger`, `--warning`, `--info` הכלליים המוגדרים ב-`:root`, אלא אם הוגדרו מחדש.
- ערכות Ocean ו-Forest הן ערכות **צבעוניות עזות** - רקע gradient מ-primary ל-secondary (כחול עז / ירוק עז) עם טקסט לבן.
- קובץ `webapp/static/css/dark-mode.css` משתמש במשתנים האלה עבור כל הרכיבים (כפתורים, קלטים, כרטיסים, מודלים, גלילה ועוד), כך ששינוי בפלטה משתקף אוטומטית בממשק.
- אם נדרש להוסיף ערכת נושא חדשה, מומלץ לעדכן קודם את הטבלה כאן ורק אז את המשתנים ב-`base.html`, כדי לשמור על תיעוד מסודר.
