## 🎨 עורך ותצוגת קוד

### 🔗 קישורים חכמים בקוד
**תיאור:** זיהוי אוטומטי של URLs, paths, ו-imports בקוד + הפיכתם לקישורים לחיצים. כולל תמיכה ב-relative imports וקישורים לקבצים אחרים במערכת.  
**קטגוריה:** Editor Enhancements  
**מורכבות:** בינונית

### 🎯 קפיצה מהירה להגדרה (Jump to Definition)
**תיאור:** Ctrl+Click על שם משתנה/פונקציה יקפוץ למקום ההגדרה שלו בקובץ או יציע חיפוש בקבצים אחרים. תמיכה בשפות פופולריות.  
**קטגוריה:** Editor Enhancements  
**מורכבות:** מורכבת

### 📐 Rulers ו-Guides ויזואליים
**תיאור:** הצגת קווים מנחים אנכיים (column rulers) ב-80/120 תווים, והדגשת זוגות סוגריים תואמים. עוזר לשמירה על תקן קוד.  
**קטגוריה:** Editor Enhancements  
**מורכבות:** קלה

### 🌈 Code Folding (קיפול בלוקים)
**תיאור:** אפשרות לקפל/לפרוס בלוקי קוד (functions, classes, loops) בלחיצה. שומר מקום מסך ועוזר לנווט בקבצים ארוכים.  
**קטגוריה:** Editor Enhancements  
**מורכבות:** בינונית

### 🔤 סמן מרובה (Multi-Cursor)
**תיאור:** עריכה בו־זמנית במספר מיקומים בקוד (Alt+Click או Ctrl+D לבחירה מרובה). מזרז עריכות חוזרות.  
**קטגוריה:** Editor Enhancements  
**מורכבות:** בינונית

### 🧩 Snippet Expansion מובנה
**תיאור:** הקלדת קיצורי דרך (כמו `for` → `for i in range():`) עם Tab completion. כולל ספריית snippets לפי שפה.  
**קטגוריה:** Editor Enhancements  
**מורכבות:** בינונית

### 📸 Code Screenshots מעוצבים
**תיאור:** יצירת תמונות מעוצבות של קוד (כמו Carbon.sh) ישירות מהעורך עם ברירות נושא, גופנים וגבולות.  
**קטגוריה:** Visualization  
**מורכבות:** בינונית

### 🖼️ תצוגת Thumbnail לקבצי תמונה
**תיאור:** הצגת preview קטן של תמונות PNG/JPG/SVG בתוך כרטיס הקובץ ברשימת הקבצים (במקום רק אייקון).  
**קטגוריה:** Visualization  
**מורכבות:** קלה

## 📄 כלי Markdown

### 📋 Table of Contents Generator
**תיאור:** יצירה אוטומטית של תוכן עניינים מכותרות המארקדאון עם עוגנים מקושרים. ניתן להציב בתחילת הדף או כ-sidebar.  
**קטגוריה:** Markdown Tools  
**מורכבות:** קלה

### ✅ Markdown Checklist State Sync
**תיאור:** שמירת מצב תיבות סימון (`- [ ]`) ב-DB כדי שתישאר מסומנת גם אחרי רענון/עריכה. שימושי ל-task lists.  
**קטגוריה:** Markdown Tools  
**מורכבות:** בינונית

### 🎨 Color Picker ל-Markdown
**תיאור:** הוספת syntax מיוחד לציון צבעים (`{color:#ff0000}`) והצגתם כ-color swatch בתצוגה. שימושי לתיעוד עיצוב.  
**קטגוריה:** Markdown Tools  
**מורכבות:** קלה

### 🖊️ Drawing/Sketching Embed
**תיאור:** הטמעת לוח ציור קטן (Excalidraw-like) בתוך מארקדאון לשרטוטים ודיאגרמות מהירות.  
**קטגוריה:** Markdown Tools  
**מורכבות:** מורכבת

### 📊 Markdown Tables Editor
**תיאור:** ממשק WYSIWYG קטן לעריכת טבלאות markdown במקום עריכת טקסט ידנית. הוספת שורות/עמודות בלחיצה.  
**קטגוריה:** Markdown Tools  
**מורכבות:** בינונית

### 🔗 Wiki-Style Links בין קבצי MD
**תיאור:** תמיכה בקישורים מסגנון ויקי (`[[קובץ-אחר]]`) בין מסמכי markdown עם autocomplete לשמות קבצים.  
**קטגוריה:** Markdown Tools  
**מורכבות:** בינונית

### 📑 Markdown Export ל-PDF/HTML
**תיאור:** ייצוא מסמך מארקדאון כ-PDF מעוצב או HTML standalone עם הנושא הנוכחי. כפתור בסרגל הכלים.  
**קטגוריה:** Markdown Tools  
**מורכבות:** בינונית

## 📊 Observability ומעקב

**תיאור:** וידג'ט קטן בדשבורד המציג CPU/Memory/Active Users בזמן אמת עם mini-sparklines. ניתן להרחיב למסך מלא.  
**קטגוריה:** Observability  
**מורכבות:** בינונית

### 🔔 Alert Preview Panel

**קטגוריה:** Observability  
**מורכבות:** בינונית

### 📊 API Response Time Histogram
**תיאור:** היסטוגרמה אינטראקטיבית (Chart.js) של זמני תגובה לכל endpoint. ניתן לסנן לפי תקופה ו-status code.  
**קטגוריה:** Observability  
**מורכבות:** בינונית

### 🕐 Time Travel Debugger (Incident Replay)
**תיאור:** הרחבה לעמוד Observability Replay עם timeline slider לסריקה אחורה בזמן ומעבר בין state snapshots של המערכת.  
**קטגוריה:** Observability  
**מורכבות:** מורכבת

### 🎯 Health Check Status Page
**תיאור:** דף פשוט ופומבי (אופציונלי) המציג סטטוס שירותים עם צבעים (ירוק/צהוב/אדום) + uptime percentage. בסגנון status.io.  
**קטגוריה:** Observability  
**מורכבות:** בינונית

### 🔍 Request Tracing Visualizer
**תיאור:** מפת טופולוגיה של request flow בין services עם זמני תגובה לכל hop. שימושי לאיתור bottlenecks.  
**קטגוריה:** Observability  
**מורכבות:** מורכבת

### 📉 Error Rate Sparklines בכרטיסים
**תיאור:** הצגת mini-sparkline בצד כל service בדף Observability שמראה מגמת שגיאות ב-24h האחרונות.  
**קטגוריה:** Observability  
**מורכבות:** קלה

**מורכבות:** קלה

**תיאור:** ממשק ויזואלי לבניית נושא מותאם אישית (בחירת צבעים, פונטים, radius) עם live preview. שמירה והחלה.  
**קטגוריה:** UI  
**מורכבות:** בינונית

### 🖱️ Right-Click Context Menu
**תיאור:** תפריט הקשר מותאם לפי מיקום (על קובץ, על פתק, על קוד) עם פעולות מהירות כמו copy, delete, share, open in new tab.

**מורכבות:** בינונית

### 🔖 Pinned Files Bar
**תיאור:** שורת קבצים מועדפים/ממוקדים בחלק העליון של הדף (כמו browser tabs) למעבר מהיר בין קבצים פתוחים.  
**קטגוריה:** UI  
**מורכבות:** בינונית

### 🎚️ Split Panes (מסך מפוצל)
**תיאור:** אפשרות לפתוח שני קבצים זה לצד זה בחלון אחד (vertical split). שימושי להשוואה ידנית או עריכה במקביל.  
**קטגוריה:** UI  
**מורכבות:** בינונית

### 🎭 Focus Mode עם פילטר צבע
**תיאור:** מצב focus שמכהה את הרקע ומדגיש רק את התוכן + אפשרות להחיל פילטר חם (sepia/amber) להפחתת עייפות עיניים.  
**קטגוריה:** UI  
**מורכבות:** קלה

### 📱 Mobile Bottom Navigation
**תיאור:** סרגל ניווט תחתון במובייל (דביק) עם אייקונים לפעולות נפוצות: חיפוש, העלאה, הגדרות, דשבורד.  
**קטגוריה:** UI  
**מורכבות:** קלה

### 🖼️ Gallery View לקבצי תמונה
**תיאור:** תצוגת גלריה (grid) לקבצי תמונות עם lightbox ל-zoom. ניווט בחיצים, הורדה, מטאדאטה.  
**קטגוריה:** UI  
**מורכבות:** בינונית

### 🎬 Animated Page Transitions
**תיאור:** אנימציות חלקות (fade/slide) במעבר בין דפים. ניתן לכבות בהגדרות למי שמעדיף פשטות.  
**קטגוריה:** UI  
**מורכבות:** קלה

## ⚡ ביצועים וזמינות

### 💾 Incremental Loading (Virtualization)
**תיאור:** ברשימת קבצים ארוכה, טעינת רק הפריטים הנראים (virtual scrolling) במקום כל ה-DB בבת אחת. מאיץ דפים גדולים.  
**קטגוריה:** Performance  
**מורכבות:** בינונית

### 📦 Background Sync ל-PWA
**תיאור:** סנכרון אוטומטי בבקע כש-PWA חוזר אונליין (Background Sync API). עדכונים שלא נשמרו יישלחו אוטומטית.  
**קטגוריה:** Performance  
**מורכבות:** בינונית

### 🔄 Smart Refresh (Pull to Refresh)
**תיאור:** משיכה כלפי מטה במובייל מרעננת את הדף (כמו באפליקציות native). עם אנימציה חלקה.  
**קטגוריה:** UI  
**מורכבות:** קלה

### 📊 Lazy Load Images עם BlurHash
**תיאור:** טעינת תמונות עם placeholder מטושטש (BlurHash) שמוחלף בתמונה האמיתית כשהיא נטענת. חוויה חלקה יותר.  
**קטגוריה:** Performance  
**מורכבות:** בינונית

### ⚡ Preload Links (Link Prefetching)
**תיאור:** preload אוטומטי של קישורים כשהעכבר מרחף מעליהם (Instant.page style). הדף הבא נטען מהר יותר.  
**קטגוריה:** Performance  
**מורכבות:** קלה

### 🌐 Offline File Queue
**תיאור:** תור עדכונים שנעשו בזמן offline (עריכות, פתקים) שיישמרו אוטומטית ברגע החזרה לאינטרנט.  
**קטגוריה:** Performance  
**מורכבות:** בינונית
